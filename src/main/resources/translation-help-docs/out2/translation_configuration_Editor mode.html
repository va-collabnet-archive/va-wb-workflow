Translation configuration - Editor mode

Editor mode:

In this section, the project manager witl be able to specify:

  • Full editor: allows any kind of translation work, adding or modifying FSN, preferred term or synonyms. If FSN term mode is selected, default term type “fully specified name” will be displayed in the translation interface. When “Preferred term mode” is chosen, default term type will be “synonym” with default acceptability “preferred”.
  • Preferred term editor: enters automatically into preferred term editing mode, and all other functions are disabled.
  • Synonyms editor: enters automatically into synonyms editor mode, and all other functions are disabled.

(NOTE: for this and all windows in this function, the specifications will only be applied upon clicking “Apply” button. Once the project manager has specified all his/her preferences, he/she must click Accept button.)
