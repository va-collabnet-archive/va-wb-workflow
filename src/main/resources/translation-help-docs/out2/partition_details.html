Partition details

This 3-tabbed panel displays partition details, partition members and worklists:

Partition tab:

The partition details tab allows the creation of worklists by clicking “Generate worklist” button.

A new window requesting to specify a worklist name will be opened:

Then, a workflow must be specified:

Finally, translation roles must be selected according to the specified workflow. More than one role may be selected (in this case, one of the selected roles will be marked as default, see both columns outlighted in red).

When "Finish" button is pressed, the worklist will be created as specified and a creation message will be displayed when the process is completed.

Members tab

This tab displays the number of Partition members. It also allows to remove one or more partition members by clicking the 'Retire selected members' button. Removed members will be visible again in the Refset Partitioner.

Worklists tab:

This tab lists the worklists that have been created.
