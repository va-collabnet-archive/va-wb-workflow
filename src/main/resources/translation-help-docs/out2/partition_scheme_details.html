Workset details - Creation of partition schemes

Worksets may be too large to be delivered to only one translator. Also, depending on each translation team workflow, some members of the translation team will translate almost any concepts, while others will review already translated concepts or review the source concepts appropriateness. Still others may review groups of concepts for establishing editorial policies and translation consistency, or there may be a specialized group of translators that master the traslation of some specific areas (e.g. odontology concepts, concepts related to medicial specialties [e.g. neurosurgery], etc.). Therefore, the project manager must have a tool for dividing the worksets of concepts to be translated according to different criteria.

When workset creation is complete, the project manager may choose to divide it in partitions. To do so, he/she should press the 'New partition scheme button' in the workset details panel.

A partition scheme is a way of dividing the workset. A given workset may be divided in several ways for different purposes and each partition criterion will be represented by a partition scheme

There cannot be two partitions with the same name. If trying to do so, a warning message will appear. However, when a partition scheme has been retired, it will be possible to create a new partition scheme using the same name as the one that was retired.

If there is no need to divide the workset into parts, the project manager may choose to create what is known as the 'One click partition'. This partition will have the same number of members as the whole workset.

When the partition scheme has been created, it is displayed in the tree view on the left under the Partition Schemes icon:

Its details will be shown in the right panel. Pressing the 'Edit partitions' button in this panel will allow the project manager to divide the workset according to specific needs .The 'Edit partitions' button opens the Refset Partitioner, which will be used to specify partitioning criteria (see the help hook in the Partition Scheme details screen).

Creation of one partition from a whole workset

A single partition from the whole workset may be created by clicking the ‘Create one click partition’ button in the Workset details panel. The One click partition will have the same number of members as the workset. Opening the ‘Members’ tab in the Partition details pane will show the list of members, as well as their number.
